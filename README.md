# 区块链记账
  1. 每个人都有记账权利
  2. 每个交易都会全网广播， 一起记录
  3. 账本每一页是一个区块
  4. 区块上有一个数 指向前一个区块
  5. 什么时候同步： 挖矿（竞争获得记账权）

# 交易
  1. 交易信息：小明 小红 100块钱
  2. 私钥加密信息（小明 小红 100块钱） sig(交易信息加了一个签名)
  地址就是公钥，可以校验sig

  矿工打包的时候，会进行每个交易的合法校验
  1. sig是签名校验合法
  2. 账号的余额够不够
